## 🚀 Project Overview

This project demonstrates how healthcare EHR data can be transformed 
from descriptive reporting into actionable NLP-driven insights 
to support predictive and operational decision-making.

## Data Source

This project uses synthetic EHR data generated using Synthea.

Synthea is an open-source synthetic patient generator that produces 
realistic but non-identifiable healthcare data.

Data files are not included in this repository.
